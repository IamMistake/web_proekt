# Application Functionalities

## Core Platform
- Product catalog with list and detail views.
- Search and comparison tools for products.
- Shopping cart and checkout flow for customers.
- Authentication for login and registration.

## Roles and Access
- Guest access to demo product listings and view-only screens.
- Customer access to cart management and order placement.
- Admin access to manage products, categories, features, suppliers, orders, and statistics.

## Admin Routes
- Product management: list, add, and edit products.
- Catalog management: categories, features, and suppliers.
- Order management and reporting/statistics.

## Maintenance and Documentation
- Database maintenance endpoints for clearing and seeding data.
- Swagger documentation for API discovery.
